# Powermind Games and Quizzes HTML Files Roadmap

## Naming convention

- **Home games**  
  *There are 2 home games per lesson.*  
  **Format:** `hg-01.01-game01.html`

- **In-class games**  
  *There is 1 in-class game per lesson.*  
  **Format:** `icg-01.01-game01.html`

- **In-class quizzes (Explore + Practice)**  
  *Unlimited number per lesson.*  
  **Format:** `icg-01.01-quiz01.html`

- **In-class reflection quizzes**  
  *Unlimited number per lesson.*  
  **Format:** `icg-01.01-rquiz01.html`
