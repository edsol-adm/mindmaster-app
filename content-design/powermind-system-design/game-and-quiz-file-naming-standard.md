- for in-class quizzes (explore+practice section) in in-class-game unit 1 lesson 1 (there is no limit on the number of quizzes): icg-01.01-quiz01.html
- for in-class quizzes (reflection section) in in-class-game unit 1 lesson 1 (there is no limit on the number of quizzes): icg-01.01-rquiz01.html
