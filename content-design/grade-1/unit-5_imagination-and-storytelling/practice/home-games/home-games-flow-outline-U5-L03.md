### Play
**Game Structure (5 problem mazes):**  
Each maze represents a real-life problem with multiple solution paths. Players must explore different routes to understand that flexibility creates resilience.

**Maze Mechanics:**  
- Top-down view of maze (bird's eye perspective)

   - Character reaches solution endpoint  
   - Success animation plays (confetti, sparkles)

   - Message reinforces that this solution works  
   - Solution details saved

   - "Try Another Path" button appears  
   - "Next Problem" button appears (if all tokens collected)  
   - "View All Solutions" button (shows solutions found so far)  
8. **Maze Restart:**  
   - Character returns to START  
   - Previously explored paths shown in lighter color

   - Collected tokens remain collected (no need to recollect)  
9. **Progress Tracking:**  
   - Each maze displays: "Solutions Found: X/Y"  
   - Visual maze map in corner highlights:  
     - Paths taken (gray)  
     - Paths not yet explored (colored/glowing)  
     - Tokens collected (checkmarks)
